# Game Icons Callout Assets

This directory contains a small sanitized subset of SVG icons from Game-icons.net for wiki callout styling.

Source: https://game-icons.net and https://github.com/game-icons/icons

License: Creative Commons Attribution 3.0 Unported, https://creativecommons.org/licenses/by/3.0/

The SVGs were sanitized for CSS mask use by removing the default background and changing the foreground fill to `currentColor`.
